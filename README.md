# movie-industry-powerbi-dashboard
Interactive Power BI dashboard analyzing 4,800+ movies to explore revenue trends, rating distribution, and budget vs box office performance.
GLOBAL MOVIE REVENUE & RATING ANALYSIS DASHBOARD

Project Overview
This project is an interactive Power BI dashboard created to analyze a dataset of more than 4,800 movies. The purpose of this dashboard is to explore trends in movie production, revenue performance, profitability, and audience ratings. The dashboard helps users quickly understand the performance of the movie industry using visual analytics.

Tools and Technologies Used
Power BI – Dashboard creation and visualization
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Creating calculated metrics and KPIs
Excel / CSV Dataset – Source data

Dataset Description
The dataset contains information about movies such as:

* Movie title
* Release year
* Budget
* Revenue
* Popularity score
* Audience rating
* Runtime

After cleaning the dataset, approximately 4,800 valid movie records were used for analysis.

Data Preparation
The dataset was cleaned using Power Query. Several unnecessary or complex columns were removed. Data types were corrected and invalid records with missing or zero values for important fields like revenue and budget were filtered out. Additional calculated fields such as profit and release year were created for better analysis.

Key Performance Indicators (KPIs)
The dashboard includes four key metrics that summarize the dataset:

Total Movies Analyzed – 4,806 movies
Total Box Office Revenue – $395.1 Billion
Total Industry Profit – $255.6 Billion
Average Movie Rating – 6.11 / 10

These KPIs provide a quick overview of the movie industry's performance.

Dashboard Visualizations

1. Top 10 Highest Grossing Movies
   This chart highlights the movies that generated the highest box office revenue. It helps identify blockbuster films that contributed the most to industry revenue.

2. Budget vs Revenue Performance (Scatter Plot)
   This visualization compares movie budgets with their box office revenue. It helps analyze whether higher production budgets lead to higher earnings and identifies outliers where low-budget movies performed exceptionally well.

3. Movie Production Trend
   This line chart shows how many movies were released each year. It helps identify trends in movie production and growth in the film industry over time.

4. Movie Rating Distribution
   This chart displays the distribution of audience ratings for movies. It shows how many movies fall into different rating ranges and helps understand overall audience reception.

Interactive Features
The dashboard includes interactive filters that allow users to explore the data dynamically.

Release Year Filter – Allows users to analyze movies released in specific years.
Rating Filter – Allows users to filter movies based on audience rating.

Additionally, custom report page tooltips were implemented so that detailed information about a specific movie appears when hovering over data points.

Key Insights from the Dashboard

* Movie production increased significantly after the year 2000.
* A small number of blockbuster movies generate a large portion of total revenue.
* Higher budgets often lead to higher revenue, but there are also successful low-budget movies.
* Most movies receive ratings between 6 and 7.

Project Outcome
This project demonstrates the use of Power BI for data cleaning, analysis, and interactive visualization. It shows how data can be transformed into meaningful insights to support data-driven decision making.

The dashboard can be useful for analysts, researchers, or anyone interested in understanding trends in the global movie industry.
